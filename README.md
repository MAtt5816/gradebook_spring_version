## Tech stack
* Java 21
* Spring Boot 3.2.3
* Thymeleaf
* Hibernate
* H2
* Preline UI
* DaisyUI
* Tailwind CSS

## How to run (dev)
1. Open terminal.
2. Type `npm run dev`
3. Build & run project.

## How to run (production)
1. Open terminal.
2. Type `npm run prod`
3. Build & run project.
